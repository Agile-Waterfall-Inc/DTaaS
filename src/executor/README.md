# Executor Application

## Getting Started

1. Install prerequisites
  - [Node v17.7.0](https://nodejs.org/en/download/current/)
2. Install dependencies

```bash
npm install
```

3. Start dev setup

```bash
npm run dev
```

4. Start application

```bash
npm start
```


## Tests

The tests can be executed by running:

```bash
npm run test
```
